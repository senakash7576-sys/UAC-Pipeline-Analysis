# Child Care Pipeline Efficiency Analysis

This project analyzes the efficiency of child care transitions from CBP to HHS using real-world data.

## Key Features
- Transfer Efficiency Analysis
- Backlog Detection
- Throughput Calculation
- Interactive Dashboard (Plotly + Jupyter)

## Tools Used
- Python
- Pandas
- NumPy
- Plotly
- ipywidgets

## Insights
- Identified bottlenecks in transfer process
- Observed backlog accumulation trends
- Analyzed system efficiency over time
